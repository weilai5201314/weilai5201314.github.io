### 三、分类和预测

#### 1.决策树

**求**：选择一堆属性，谁为根属性？

```
墒：无条件，有条件
信息增益
```

答：先算无条件墒，再算条件墒，然后求信息增益，决定根属性。

**问**：如何求叶节点？

答：重复根属性的获取方式，直到最后一个。

⚠️：一个根属性的不同分支，信息增益不同，要按照根属性来划分计算。

```
例：
			根属性划分				第二次的划分属性
			
		=>  30<			   =>工作
年龄 =>  30~40			=>信誉
		=>	>40				 =>地位
		
每一次划分之后都要重新计算全部，确定最优属性。
```
